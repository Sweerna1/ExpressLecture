# Code Task

## Setting Up the Project

1- Initialize a new Node.js project.

2- Install Express.

<br>

## Creating Endpoints

1- Create GET endpoint that returns list of cakes objects.

2- Create GET endpoint that returns single cake object.

3- Create POST request to add new cake object.

3- Create PUT request to update a cake object.

4- Create DELETE request to delete a cake object.

<br>
Notes:

- Make sure to import all the needed dependencies.
- Import Nodemon for better performance.
- Use Joi node package for input validation.
